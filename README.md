# 📄 宣传资料二维码

扫码查看最新宣传资料，内容随时更新。

## 使用方法

### 更新资料（无需AI，无需命令行）

1. 打开浏览器，进入 GitHub 仓库页面
2. 点进 `content/` 文件夹
3. 点右上角 **"Add file" → "Upload files"**
4. 把新的 PDF 文件**拖进去**
5. 往下滑点 **"Commit changes"**（绿色按钮）
6. 等 2-3 分钟，扫码就能看到新内容

### 在页面上添加新文件

上传 PDF 后，还要在 `index.html` 的文件列表里加一行：

1. 在仓库首页点 `index.html`
2. 点右上角的 ✏️ 编辑按钮
3. 找到下面这段代码：

```javascript
const PDF_FILES = [
    // ↓↓↓ 从这里开始加文件 ↓↓↓
    // { name: "示例宣传册", file: "example.pdf" }
    // ↑↑↑ 加在这里 ↑↑↑
];
```

4. 在中间加一行，比如：
   ```javascript
   { name: "2025年新品介绍", file: "新品介绍.pdf" },
   ```
5. 点 **"Commit changes"** 保存

> **注意：** 每上传一个新 PDF，都要在 index.html 里加一行。文件名要写对。

### 维护好习惯

- PDF 文件名用**中文或拼音**，不要有特殊符号
- 文件名和 `index.html` 里写的文件名要**一模一样**
- 不需要的文件可以直接在 GitHub 上删除
